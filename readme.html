Getting Started with Conversate

Conversate is a twitter-style WordPress theme that features easy posting from the front page and automatic updates for posts.

Skip to Installation Instructions

Features

Conversate offers some robust but easy-to-use features:

  • 5 Color Schemes

    Choose from 5 different color palettes and upload your own logo and background image to customize Conversate in just minutes.

  • Inline Comment Posting & Editing

    Post and edit comments directly and instantly.

  • Handy Built-In Keyboard Shortcuts

    Use the keys on your keyboard to quickly scroll through all the updates on the page or even post a quick comment reply! Just press the 'h' key to see all the available commands!

  • Post videos, MP3 files, and More

    Share videos and music with others through the quick post form on the homepage.

  • iPhone Styles Included

    Conversate includes a built-in iPhone stylesheet, making it easy to browse and post updates on the go.

  • Threaded Comments

    Conversations are made easy with threaded comments.

  • Live Tag Suggestion

    Makes tagging posts easier by offering similar tags used in the past.

  • Real-Time Notification with Comments or Posts

    Get notified instantly when a new comment or post is created.

  • Includes the UpThemes Framework

    Upload images, update colors, and change layout options and more! The UpThemes Framework makes it super easy to customize your theme.

Installing Conversate

How to Install

Install Conversate Theme Through WordPress:

  1. Log into WordPress through the admin panel. (i.e. www.yourdomain.com/wp-admin)
  2. Go to the "Appearance" module in the left navigation and select "Themes"
  3. You should see two tabs "Manage Themes" and "Install Themes", Click on "Install Themes".
  4. Now click on "Upload" in the sub nav.
  5. Select the Conversate Theme ZIP file located on your local machine.
  6. Click Install.
  7. After your theme installs you should get a confirmation that it was successful, you will have the option to "Activate", "Preview", or "Return to Themes Page".
  8. Click "Activate".
  9. Success! Your new UpThemes Conversate Theme Theme is now active.

Install Conversate Theme Manually:

  1. Upload the Conversate Theme folder to the themes folder located on your server. (i.e. wp-content/themes/)
  2. Once this is completed log into WordPress through the admin panel. (i.e. www.yourdomain.com/wp-admin)
  3. Go to the "Appearance" module in the left navigation and select "Themes".
  4. You should see two tabs "Manage Themes" and "Install Themes" click on "Manage Themes".
  5. In this tab you will see "Current Theme" which is the standard WordPress theme or perhaps another theme you had installed previously. Also,"Available Themes" which are the list of themes that you have uploaded to your server. You should see Conversate Theme 1.0 in the available themes section. Click "Activate".
  6. Success! Your new UpThemes Conversate Theme Theme is now active.

Additional Instructions

Install WP-PageNavi

In order for your installation of Conversate to have the upgraded page navigation styles at the bottom of the page, you will need to download and install the WP-PageNavi plugin.
